# WBS出力 ユーザー仕様

最終更新: 2026-07-03
参照元: `docs/01要件定義・設計/要件定義書.md` / `docs/01要件定義・設計/基本設計書.md` / `DESIGN.md` / `AGENTS.md`

## 概要

トップバーの「WBSに上書き」から、スプレッドシートの `WBS` シートを自動生成します。

## 実行条件

- 未同期の保存がある間は実行できません。
- 実行前に確認モーダルを表示します。
- 完了すると「WBSシートを更新しました」と表示します。
- 日付範囲が400日を超える場合は、400日で切り詰めた警告を表示します。

## 出力内容

- シート上部に自動生成であることの注記を出します。
- メタ情報、マイルストーン、会議体、WBS表、日付グリッドを出力します。
- 成果物、備考、会社名、責任者、計画日、実績日、進捗率、完了フラグを出力します。
- 計画期間は淡色のバー、実績期間は `★`、マイルストーンは `▼` で表示します。

## WBS対象外

- タスク詳細の「WBSに反映」をOFFにしたタスクは、子タスクごと出力されません。
- 親がWBS対象外の場合、子がONでも出力されません。
- アプリ内の看板・リスト・ガントでは、対象外タスクに「WBS外」チップを表示します。

## 注意

- アプリは `WBS` シートを読みません。
- `WBS` シートを直接手編集しても、次回出力で上書きされます。
