# ステータス列管理 ユーザー仕様

最終更新: 2026-07-16
参照元: `docs/01要件定義・設計/要件定義書.md` / `docs/01要件定義・設計/基本設計書.md` / `DESIGN.md` / `AGENTS.md`

## 目的

ステータス列は看板のレーンであり、タスクの進捗状態を表します。

## 初期列

- 未着手。
- 進行中。
- 完了。

## 列追加

- 看板ビューの「列を追加」ボタンから追加できます。
- 名称、色、完了列にするか、親の自動集計で使う進行中列にするかを設定できます。
- 色はプリセットが基本で、詳細オプションから自由色を選べます。

## 列編集

- 各列ヘッダーの編集ボタンから編集できます。
- 名称変更、色変更、完了列切替、進行中列切替ができます。
- 完了列に設定できる列は常に1つだけです。
- 進行中列の役割は列名と分離されるため、「レビュー中」などへ名称変更しても親の自動集計は維持されます。
- 完了列と進行中列を同じ列には設定できません。

## 列削除

- 列編集画面から削除できます。
- 削除対象列にタスクが残っている場合は削除できません。
- 完了列と進行中列は削除できません。先にそれぞれの役割を別の列へ切り替えてください。

## 並び替え

- 列ヘッダーをドラッグして左右順を変更できます。
- 並び順は保存され、次回読み込み時にも維持されます。
