## ローカルキャッシュ

- キーは `gasTaskManager.cache.v1` です。
- `snapshotForCache()` がnodes/members/statusColumns/dependencies/commentCounts/currentMember/rootIdを保存します。
- setupRequired状態はキャッシュしません。

## 自動同期

- 間隔は `SYNC_INTERVAL_MS = 30000` です。
- 保存中ノードがある場合は `syncStatus = pending` にして同期を中断します。
- 強制同期は `syncFromServer({ force: true })` です。

## 保存キュー

- `state.saveChains` はNodeIdごとのPromiseチェーンです。
- `state.latestRequestByNode` は最後に発行したrequestIdを保持します。
- 応答requestIdが最新でない場合、結果を適用しません。
- optimisticApply後はローカルキャッシュにも保存します。

## 楽観的競合検知

- Nodes系保存だけ `baseUpdatedAt` を照合します。
- `saveNode()` / `moveNode()` / `fitNodeToChildren()` が対象です。
- Members、StatusColumns、DependenciesはUpdatedAtを持たず、LockServiceによる直列化のみです。
