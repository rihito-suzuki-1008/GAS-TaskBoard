# マイルストーン・会議体 ユーザー仕様

最終更新: 2026-07-04
参照元: `docs/01要件定義・設計/要件定義書.md` / `docs/01要件定義・設計/基本設計書.md` / `DESIGN.md` / `AGENTS.md`

## 開き方

- トップバーの「設定」を押します。
- プロジェクト設定モーダルで「マイルストーン」「会議体」タブを切り替えます。

## マイルストーン

- 名称、日付、備考を登録できます。
- 日付はアプリ内カレンダーで選択します。
- 登録したマイルストーンはWBS出力上部と日付グリッドに表示されます。

## 会議体

- 名称、開催ルール、備考を登録できます。
- 開催ルールは、指定なし、毎日、毎週、隔週、毎月(日付)、毎月(曜日)から選べます。
- 毎日/毎週/隔週/月次のルールでは初回開催日を設定します。終了日は任意です。
- 登録した会議体はガントビューとWBS出力上部に表示されます。
- WBS出力の日付グリッドには開催日の `▼` だけを表示し、会議名は左側の行に表示します。

## 編集・削除

- 一覧の編集ボタンでフォームに読み込みます。
- 削除は確認モーダルを経由します。
