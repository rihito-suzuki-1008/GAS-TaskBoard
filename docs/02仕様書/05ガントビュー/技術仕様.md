## スケール計算

- `ganttScale(rows)` は表示対象の `displayStartDate` と `displayEndDate` から全体範囲を決めます。
- 前後に余白日数を追加します。
- `state.ganttZoom` に応じてbaseDayWidthを変えます。
- `state.ganttScaleFactor` を乗算し、横の縮尺スライダーを反映します。

## 作成・日程描画レーン

- 左ツリーにはリストビューと同じ「＋」とゴースト行を表示し、子shell作成は `createChildShell(parentId)` に統一します。
- タイムライン背景は行の状態で動作を分けます。
  - 日付未設定の末端ノード行は `data-gantt-schedule-node` を持ち、pointerupで `saveNodePatch(nodeId, { startDate, endDate })` を呼び、そのノード自身のバーを描きます。
  - 子を持つノード行、ゴースト行、空レーンは `data-gantt-create-parent` を持ち、pointerupで `createChildShell(parentId, { startDate, endDate })` を呼びます。
  - 日付設定済みの末端ノード行は背景ドラッグ対象にしません。
- クリック（ドラッグ距離が閾値未満）も、その日1日のバー作成またはshell作成として扱います。
- pointermove中は日境界にスナップしたプレビューと日付チップを表示します。
- Escキーでドラッグをキャンセルし、ドラッグ中は全体 `render()` を延期します。

## バー操作

- バーは `data-gantt-bar` を持ちます。
- ハンドルは `data-handle="left"` / `data-handle="right"` で判定します。
- pointermove中はDOM styleだけを更新し、pointerupで `saveNodePatch()` を呼びます。
- 自身の日程がない親タスクは表示期間を持っていても、保存対象日程としては扱いません。

## 依存矢印

- 表示中の行にない依存先は、表示中の祖先にアンカーします。
- SVG pathで折れ線を描画します。
- 開始/終了座標は日付からdayWidth単位で計算します。
