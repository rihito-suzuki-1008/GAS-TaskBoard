## 関連ファイルと関数

- `src/ClientRenderViews.html`: `renderGantt()`, `renderGanttBar()`, `renderGanttArrows()`, `renderAxisTicks()`。
- `src/ClientBindings.html`: `bindGantt()`。
- `src/ClientActions.html`: `startGanttCreateDrag()`, `startGanttDrag()`。
- `src/ClientSelectors.html`: `ganttScale()`。
- `src/Styles.html`: ガントレイアウト、shellBarGrow、作成プレビュー。

## スケール計算

- `ganttScale(rows)` は表示対象の `displayStartDate` と `displayEndDate` から全体範囲を決めます。
- 前後に余白日数を追加します。
- `state.ganttZoom` に応じてbaseDayWidthを変えます。
- `state.ganttScaleFactor` を乗算し、横の縮尺スライダーを反映します。

## 作成レーン

- ルート作成レーンは `data-gantt-create-parent` にrootIdを持ちます。
- 各行背景も同属性にそのノードIDを持ちます。
- pointerdownからpointermoveでプレビューを生成します。
- pointerup時にドラッグ距離が閾値以上なら `createChildShell(parentId, { startDate, endDate })` を呼びます。

## バー操作

- バーは `data-gantt-bar` を持ちます。
- ハンドルは `data-handle="left"` / `data-handle="right"` で判定します。
- pointermove中はDOM styleだけを更新し、pointerupで `saveNodePatch()` を呼びます。
- 自身の日程がない親タスクは表示期間を持っていても、保存対象日程としては扱いません。

## 依存矢印

- 表示中の行にない依存先は、表示中の祖先にアンカーします。
- SVG pathで折れ線を描画します。
- 開始/終了座標は日付からdayWidth単位で計算します。
