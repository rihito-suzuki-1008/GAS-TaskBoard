# ガントビュー ユーザー仕様

最終更新: 2026-07-22
参照元: `docs/01要件定義・設計/要件定義書.md` / `docs/01要件定義・設計/基本設計書.md` / `DESIGN.md` / `AGENTS.md`

## 目的

ガントビューは、タスク階層と期間を横軸の日付で確認し、期間の作成・変更を直接行う画面です。

## レイアウト

- 左側にタスク名、日付範囲、日数が表示されます。
- 縦スクロール中も左側の列見出しは不透明な背景で前面に残り、下のタスク名や日付とは重なりません。
- 右側にタイムライン、月帯、日付目盛り、今日線、タスクバー、依存矢印が表示されます。
- 依存矢印はタスクバーの外側と行間を通ります。経路が重なる場合は別レーンへ分かれ、交差箇所も縁取りによって線を追いやすくなります。
- 折りたたんだグループ内の複数依存が同じ親同士の線になる場合は、重複線を1本にまとめます。
- タスク行の上に、日付設定済みのマイルストーンと会議体が専用レーンで表示されます。
- 縦スクロール中もマイルストーン/会議体の予定レーンは日付ヘッダー直下に固定され、タスク行だけがその下をスクロールします。
- マイルストーン/会議体の `▼` にホバーすると、名称、日付、開催ルールなどのガイドが表示されます。
- 近い日付のマイルストーンは横表示の密度に応じて縦レーンをずらし、重ならないようにします。
- 予定レーンはツールバーから折りたためます。
- タスク階層の開閉状態はリストビューと共通です。同じブラウザでは案件・ログインユーザーごとに保存され、次回起動時にも復元されます。
- 上部で日/週/月の表示粒度を切り替えられます。
- 横の縮尺は「− / ＋」ステッパーで段階的に調整でき、現在の倍率が中央に表示されます。
- ツールバーの「予定設定」からマイルストーン/会議体の設定を開けます。
- 「今日へ」ボタンで今日線の位置へ横スクロールできます。
- タスクがまだない未来日にも横スクロールでアクセスできます。

## 0からタスクを作る操作

- タイムライン下部の空レーンをドラッグすると、ルート直下のタスクを作成できます。
- 日程設定済みで子を持つタスク行の背景をドラッグすると、そのタスクの子タスクを作成できます。
- ドラッグした範囲が開始日と終了日になります。
- 作成後は右ドロワーが開き、名前入力にフォーカスします。
- 既存タスク行の背景はクリックだけでは作成しません。日程付き作成はドラッグ操作で行います。
- 左ツリーの子タスク追加ボタンにホバーすると、挿入位置を示す横線と「新しいタスク」チップが表示されます。
- 日程未設定のタスク行をドラッグすると、新規作成ではなく、そのタスク自身に日程を設定します。

## 期間変更

- タスクバー本体をドラッグすると期間全体を移動できます。
- 左端ハンドルをドラッグすると開始日を変更できます。
- 右端ハンドルをドラッグすると終了日を変更できます。
- 保存後、依存関係がある後続タスクは必要な分だけ自動調整されることがあります。

## アニメーション

- ガントから作成したshellバーは横方向に伸びるように表示されます。
- リストビューのshell行と同じく、作成直後の「生えてきた」感が分かるようになっています。
