# メンバー管理 ユーザー仕様

最終更新: 2026-07-02
参照元: `docs/01要件定義・設計/要件定義書.md` / `docs/01要件定義・設計/基本設計書.md` / `DESIGN.md` / `AGENTS.md`

## 目的

メンバー管理では、タスク担当者として使う人の名前、メールアドレス、表示色を管理します。

## 開き方

- 上部の「メンバー」ボタンを押します。
- メンバー管理モーダルが開きます。

## 追加・編集

- 氏名、メール、色を入力します。
- 色はプリセットから選べます。
- 詳細オプションで自由色も選べます。
- 既存メンバーの編集ボタンを押すと入力欄へ内容が読み込まれます。

## 削除

- メンバー行の削除ボタンから削除確認に進みます。
- 自分自身を削除する場合は追加の確認チェックがあります。
- 最後の1人は削除できません。
- 削除されたメンバーは担当中タスクから外れます。

## 表示への影響

- 担当者はアバターとして表示されます。
- 看板左の担当者一覧にも反映されます。
- タスク詳細の担当者選択肢にも反映されます。
